# Stage 2：微服务架构（Day 11-17）

## 📚 学习目标
掌握微服务核心组件，能搭建微服务 demo 并讲清架构原理。

## 📅 学习内容

### Day 11：微服务概念 + Nacos
- 服务注册发现核心思想
- 安装 Nacos + 服务注册实操

### Day 12：Gateway 网关
- 路由配置、跨域处理、过滤器
- 网关核心作用

### Day 13：OpenFeign 远程调用
- 服务间调用代码实现
- 调用流程讲解

### Day 14：微服务统一认证
- 网关 + JWT 整合
- 全链路鉴权流程

### Day 15：MQ 概念（面试专用）
- 异步、解耦、削峰核心价值
- 适用业务场景

### Day 16：微服务项目 demo 打通
- 服务拆分思路
- 网关+注册+调用+认证整合

### Day 17：微服务面试话术整理
- 微服务改造经验包装
- 项目讲解技巧

## 🏗️ 项目结构
```
Stage2-MicroService/

```
